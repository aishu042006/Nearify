# Nearify

Nearify is a modern MERN Stack platform that helps users discover and create opportunities in one place. Users can explore internships, hackathons, events, and food offers while also publishing their own opportunities through an intuitive and responsive interface.

## Features

### User Authentication

* Secure Login & Registration
* User Profile Management
* Logout Functionality

### Dashboard

* Personalized Dashboard
* Quick Access Navigation
* Modern and Responsive UI

### Opportunities

* Browse Internships
* Explore Hackathons
* Discover Events
* View Detailed Information

### Create Opportunity

Users can create and publish:

* Internships
* Hackathons
* Events

### Food Offers

* Explore Restaurant Offers
* View Restaurant Details
* Reservation-Friendly Interface

### Profile Management

* Update Personal Information
* Upload Profile Picture
* Upload Resume
* Manage Skills and Links

### Saved Items

* Save Opportunities
* Save Events
* Access Favorites Easily

### Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Optimized

## Installation

Clone the repository:

```bash
git clone https://github.com/aishu042006/nearify.git
```

Navigate to the project:

```bash
cd nearify
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
